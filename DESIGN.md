# 英语老师排课小程序 - 设计文档

## 产品灵魂与意象锚点

**意象锚点**：一面布满彩色便利贴的教室白板——每张便利贴代表一节课，不同颜色代表不同课程类型，错落有致地排列在时间格子里。温暖、有序、一目了然。

**气质关键词**：教育感、温暖、清晰、专业、亲和
**场景画面**：阳光透过教室窗户洒在木质桌面上，墙上的课程表用彩色马克笔清晰标注着每一节课的时间与内容。

## 视觉策略

- **摄影/图像方向**：使用柔和明亮的色彩，避免冷硬科技感
- **图形语言**：圆角卡片、温和阴影、彩色标签区分课程类型

## 组件选型原则

- **通用 UI 组件**：按钮、输入框、卡片、标签、弹窗、Tabs、Toast、分隔线等通用组件优先使用 `@/components/ui/*`
- **禁止**用 `View/Text` 手搓通用组件
- 新页面先拆解 UI 单元，每个单元映射到组件库

## 页面结构

| 页面 | 路由 | 功能 |
|------|------|------|
| 排课主页 | `pages/index/index` | 周视图日历 + 快速添加排课 |
| 教师管理 | `pages/teachers/index` | 教师列表 + 增删改 |
| 学生管理 | `pages/students/index` | 学生列表 + 增删改 |
| 课程管理 | `pages/courses/index` | 课程类型管理 |

## 导航结构

- 使用 TabBar 切换：排课主页、教师管理、学生管理、课程管理
- TabBar 图标使用 lucide-react-taro 图标

## 间距系统

- 页面边距：`p-4`
- 卡片内边距：`p-4`
- 组件间距：`gap-3` / `gap-4`
- 列表项间距：`space-y-2`

## 卡片样式

- 圆角：`rounded-2xl`
- 阴影：`shadow-sm`
- 背景：`bg-white`

## 跨端约束

- 底部 TabBar 区域需避让（`bottom-12` 或 `pb-12`）
- 小程序与 H5 双端兼容
- Input 用 View 包裹
- Text 垂直排列加 `block`
